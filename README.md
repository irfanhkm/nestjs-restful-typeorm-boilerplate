## Requirement
- NodeJS LTS
- MySQL
- ts-node

## Installation
- clone repository
- npm install
- env setup
- npm run typeorm migration:run -- -c database_connection 

## Tutorial Reference
- TypeOrm API https://typeorm.io/#/
- TypeORM Script:
  - npm run typeorm (parameter args add --)
  - example: `npm run typeorm migration:generate -- -c database_1`
